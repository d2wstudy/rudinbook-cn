# 重要符号表

下表所列符号附以简短说明。

| 符号 | 说明 | 符号 | 说明 |
|------|------|------|------|
| ∈ | 属于 | → | 收敛于 |
| ∉ | 不属于 | lim sup | 上极限 |
| ⊂, ⊃ | 包含符号 | lim inf | 下极限 |
| Q | 有理数域 | $g \circ f$ | 复合 |
| <, ≤, >, ≥ | 不等符号 | $f(x+)$ | 右极限 |
| sup | 最小上界 | $f(x-)$ | 左极限 |
| inf | 最大下界 | $f', f'(x)$ | 导数 |
| R | 实数域 | $U(P, f), U(P, f, a), L(P, f), L(P, f, a)$ | Riemann 和 |
| $+\infty, -\infty, \infty$ | 无穷大 | $\mathcal{R}, \mathcal{R}(a)$ | Riemann (Stieltjes) 可积函数类 |
| $\bar{z}$ | 复共轭 | $\parallel \parallel$ | 范数 |
| $\operatorname{Re}(z)$ | 实部 | $\mathcal{C}(X)$ | 连续函数空间 |
| $\operatorname{Im}(z)$ | 虚部 | $\exp$ | 指数函数 |
| $\|z\|$ | 绝对值 | $D_N$ | Dirichlet 核 |
| $\Sigma$ | 求和符号 | $\Gamma(x)$ | $\Gamma$ 函数 |
| $R^k$ | $k$ 维欧氏空间 | $\{e_1, \dots, e_n\}$ | 标准基 |
| $\mathbf{0}$ | 零向量 | $L(X), L(X, Y)$ | 线性变换空间 |
| $x \cdot y$ | 内积 | $[A]$ | 矩阵 |
| $\|x\|$ | 向量 $x$ 的模 | $D_i f$ | 偏导数 |
| $\{x_n\}$ | 序列 | $\nabla f$ | 梯度 |
| $\cup, \bigcup$ | 并 | $\mathcal{C}', \mathcal{C}''$ | 可微函数类 |
| $\cap, \bigcap$ | 交 | $\det[A]$ | 行列式 |
| $(a, b)$ | 开区间 | $J_r(x)$ | 函数行列式 |
| $[a, b]$ | 闭区间 | $\frac{\partial(y_1, \dots, y_n)}{\partial(x_1, \dots, x_n)}$ | 函数行列式 |
| $E^\circ$ | $E$ 的余集 | $I^k$ | $k$ -方格 |
| $E'$ | $E$ 的极限点的集 | $Q^k$ | $k$ -单形 |
| $\bar{E}$ | $E$ 的闭包 | $dx$ | 基本 $k$ -形式 |
| $\lim$ | 极限 | $\wedge$ | 乘号 |
| $d$ | 微分算子 | $\omega_T$ | $\omega$ 的变换 |
| $\partial$ | 边缘算子 | $\nabla \times F$ | 旋度 |
| $\nabla \cdot F$ | 散度 | $\mathcal{E}$ | 初等集的环 |
| $m$ | Lebesgue 测度 | $\mu$ | 测度 |
| $\mathfrak{M}_F, \mathfrak{M}$ | 可测集的类 | $\{x \mid P\}$ | 带有性质 $P$ 的集 |
| $f^+, f^-$ | $f$ 的正(负)部 | $K_E$ | 特征函数 |
| $\mathcal{L}, \mathcal{L}(\mu), \mathcal{L}^2, \mathcal{L}^2(\mu)$ | Lebesgue 可积的函数类 | | |
